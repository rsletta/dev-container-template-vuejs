# Dev container template for Vue.JS

Use this template repository to bootstrap new Vue 3 project, where you can isolate the development environment in a dev container.

## Added Container Features

- [Azure CLI](https://docs.microsoft.com/en-us/cli/azure/get-started-with-azure-cli)
- [GitHub CLI](https://cli.github.com)
